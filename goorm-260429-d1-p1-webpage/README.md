# Goorm 260429 Day 1 Project 1

Day 1 정적 랜딩 웹페이지 프로젝트입니다.

## 개요

- 형태: 단일 페이지 정적 웹앱 (`HTML/CSS/JavaScript`)
- 목적: 프런트엔드 기본 UI/인터랙션 실습
- 진입 파일: `index.html`

## 주요 기능

- 라이트/다크 테마 전환 (`localStorage` 저장)
- 다국어 전환 (`KO / EN / JA / ZH`)
- FAQ 아코디언
- 스크롤 기반 섹션 하이라이트
- 문의 폼 (`mailto:` 기반)

## 실행 방법

```bash
cd goorm-260429-d1-p1-webpage
python3 -m http.server 5500
```

- 브라우저: `http://localhost:5500`

## 파일 구성

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
