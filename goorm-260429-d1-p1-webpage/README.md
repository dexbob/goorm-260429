# Goorm 260429 Day 1 Project 1 - Webpage

구름 260429 과정의 Day 1 정적 웹페이지 프로젝트입니다.

## 프로젝트 개요
- 형태: 단일 페이지 정적 웹앱 (`HTML/CSS/JavaScript`)
- 목적: 바이브코딩/프런트엔드 기본 구성 실습
- 진입 파일: `index.html`

## 주요 기능
- 라이트/다크 테마 토글 (`localStorage` 저장)
- 다국어 전환 (`KO / EN / JA / ZH`)
- FAQ 아코디언 UI
- 스크롤 기반 섹션 하이라이트
- 문의 폼 (`mailto:` 기반 전송)

## 실행 방법
```bash
cd goorm-260429-d1-p1-webpage
python3 -m http.server 5500
```

브라우저 접속:
- `http://localhost:5500`

## 파일 구성
- `index.html`
- `styles.css`
- `script.js`
- `assets/`
