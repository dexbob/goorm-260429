페이지 열기 루트로 이동

데이터 분석 자동화 AI 에이전트

CSV 파일을 업로드하면 자동으로 EDA/시각화/인사이트를 생성하는 React + FastAPI 프로젝트입니다.

핵심 구성

  • 프론트엔드: Vite + React
  • API: FastAPI (api/index.py)
  • 분석 파이프라인: python/pipeline.py
  • 노트북 생성: python/build_notebook.py

로컬 실행

포트 자동 탐색 오케스트레이터를 사용합니다.

npm run dev
  • API(uvicorn)와 프론트(Vite)를 동시에 실행
  • 사용 포트는 .runtime-ports.json로 공유

Vercel 배포

  • Root Directory: goorm-260506-d5-p1-webpage
  • Build Command: npm run build
  • Install Command: npm ci
  • Output Directory: dist

API는 same-origin /api/*를 사용합니다. 외부 백엔드를 따로 붙일 때만 VITE_API_BASE_URL을 지정하세요.
