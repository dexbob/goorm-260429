# Goorm 260501 Day 3 Project 2 - 습관제조기

횟수 기반 습관 추적 웹앱입니다.  
하루 목표 횟수를 설정하고, 오늘 진행도를 배터리형 버튼으로 기록하며, 연도별 GitHub 스타일 히트맵으로 누적 기록을 확인할 수 있습니다.

## 개요

- 형태: 정적 웹앱 (`HTML/CSS/Vanilla JavaScript`, ES Modules)
- 진입 파일: `index.html`
- 데이터 저장: 브라우저 `localStorage` (`habit-maker-v1`)
- 대상 환경: 최신 브라우저 (모듈 스크립트 지원)

## 주요 기능

- 습관 추가
  - 이름, 아이콘, 목표 횟수(1~10 슬라이더), 색상 선택
  - 선택 색상에 맞춘 버튼/슬라이더 UI 연동
- 진행 기록
  - 배터리형 버튼 클릭 시 오늘 횟수 +1
  - 목표 달성 후 추가 클릭 시 "오늘 진행도를 초기화할까요?" 확인 후 0으로 초기화
- 습관 삭제
  - 카드 아이콘 버튼 클릭 시 확인 팝업 후 삭제
- 히트맵 시각화
  - 연도별(1월 1일~12월 31일) GitHub 스타일 셀 렌더링
  - 월/요일(한국어) 라벨, 연도 이동 버튼, 색상 강도 단계 표시
  - 각 습관 카드 내부에서 가로 스크롤 지원
- 레이아웃/테마
  - 다크 테마 기반 UI
  - 상단 타이틀/입력 영역 고정, 습관 목록 스크롤

## 실행 방법

```bash
cd goorm-260501-d3-p2-webpage
python3 -m http.server 5500
```

- 브라우저: `http://localhost:5500`
- 루트에서 `./start-servers.sh` 실행 시에도 정적 프로젝트로 자동 노출됩니다.

## 파일 구성

- `index.html`: 앱 마크업, 습관 카드 템플릿, 확인 모달
- `css/style.css`: 다크 테마/폼/카드/히트맵/배터리 버튼 스타일
- `js/app.js`: 이벤트 바인딩, 상태 반영, 사용자 액션 처리
- `js/ui.js`: 카드 및 히트맵 렌더링
- `js/storage.js`: `localStorage` 로드/저장
- `js/habit.js`: 습관 객체 생성
- `js/record.js`: 일일 기록 조회/증가/설정
- `js/stats.js`: 진행도 계산 유틸
- `js/heatmap.js`: 히트맵 강도 계산 유틸
- `js/state.js`: 전역 상태 저장소

## 참고

- 저장 데이터 초기화가 필요하면 브라우저 개발자 도구에서 `localStorage`의 `habit-maker-v1` 키를 삭제하세요.
- 로컬 시간대 기준 날짜를 사용하도록 구현되어 히트맵 툴팁 날짜와 실제 요일 오차를 줄였습니다.
